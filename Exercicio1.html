Exercício 1

- Verificação de Número Positivo, Negativo ou Zero

1- Verificação de número positivo, negativo ou zero:

Enunciado: Desenvolva um algoritmo em PHP que receba, através de um formulário HTML, um número digitado pelo usuário e determine se o valor é positivo, negativo ou igual a zero. A resposta deve ser exibida no navegador em formato de mensagem. Saída com CSS: "Valor positivo" → classe .resultado-pos (verde) "Valor negativo" → .resultado-neg (vermelho) "Igual a zero" → .resultado-zero (azul) .

-------------------------------------

Passos a seguir:

Crie um formulário HTML aqui que envie o número para o backend PHP via método POST.

Lembre-se: form action="Exercicio1.php" method="post"

O PHP deve processar a entrada e exibir o resultado na mesma página ou em uma página separada.

Regra geral:

Session: só para mostrar quem está logado (via autenticacao.php)

Cookie: para gravar a última entrada do usuário neste exercício - Salve em cookie: setcookie('ex01_ultimo', $numero, ...). Na próxima visita, mostre: "Último número: -3"
